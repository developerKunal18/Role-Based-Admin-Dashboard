# Role-Based Admin Dashboard

A Flask web application with role-based authentication.

## Features
- User registration
- User login
- Admin/user roles
- Protected admin panel
- Secure password hashing

## Technologies Used
- Python
- Flask
- SQLite3
- bcrypt
- HTML

## Installation

pip install flask bcrypt

## Run

python app.py

Open browser:
http://127.0.0.1:5000

## Purpose
focuses on authentication and authorization systems.
